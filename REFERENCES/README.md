## E.COLI REFERENCE

#### Aquí encontramos los genomas de referencia utilizados para la anotación de los genomas y el análisis filogenético.

El genoma de referencia utilizado pertenece a la cepa ***Escherichia coli* str. K-12 substr. MG1655** con código de accesión **GCA_000005845.2**
* *reference.fna* se utiliza para la anotación con `Medaka`
* *genomic.gbff* se utiliza para el análisis filogenético con `Snippy`

Nota: El archivo *E.coli_genomes.xlxs* contiene la información de las cepas de *Escherichia coli* usadas para el análisis filogenético.
